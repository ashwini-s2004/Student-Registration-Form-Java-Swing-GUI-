# 📖 Project Overview

The **Student Registration Form** is a desktop-based Java Swing application designed to simplify the process of collecting and validating student information.

The application enables users to enter student details through an interactive graphical interface, validates mandatory fields, displays error messages when required, and presents a formatted summary after successful submission.

Built with a modern peach-themed interface and responsive layout, this project demonstrates core Java GUI programming concepts, event handling, and user-friendly application design.
