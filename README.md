# 🎓 Student Registration Form - Java Swing

## 📄 Description
A Java Swing-based Student Registration Form that collects and validates student details like name, roll number, gender, course, year, email, phone, and address. Includes **Submit** and **Clear** buttons, error handling, and a modern peach-themed responsive GUI for interactive form entry.

---

## 🧩 Features
- Input fields for:
  - Full Name
  - Roll Number
  - Gender (Male/Female/Other)
  - Email ID
  - Phone Number
  - Course selection (B.Tech, B.Sc, BCA, MCA, MBA)
  - Year selection (1st–4th Year)
  - Address
- **Submit** button: Validates input and displays a summary of entered data
- **Clear** button: Resets all form fields
- Error alerts for mandatory fields
- Responsive layout: Form is centered dynamically on window resize
- Modern, peach-themed GUI for better user experience

---

## 🛠️ Technologies Used
- **Language:** Java  
- **Framework:** Java Swing  
- **IDE:** IntelliJ IDEA / Eclipse / VS Code  
- **JDK Version:** 8 or above
